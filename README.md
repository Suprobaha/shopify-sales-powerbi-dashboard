# Shopify Sales Power BI Dashboard

## Overview
This project is an interactive Power BI dashboard built to analyse Shopify sales performance, customer behaviour, and repeat purchase trends.

## Project File
- Shopify Analysis.pbix

## Key Features
- Net Sales KPI
- Total Quantity
- Net Average Order Value
- Total Customers
- Single Order Customers
- Repeat Customers
- Lifetime Value
- Repeat Rate
- Purchase Frequency
- Regional sales analysis
- Sales by day and hour
- Product type analysis
- Payment gateway analysis

## Dashboard Pages

### 1. Shopify Analysis
Main dashboard showing KPIs, customer funnel metrics, map analysis, sales trends, and breakdowns by product type and payment gateway.

### 2. Details Table
Detailed table containing order-level and customer-level information such as order number, customer ID, city, province, product type, gateway, tax, and total price.
